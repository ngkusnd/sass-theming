    flatten

    @function flatten($map, $separator, $_parent-key) { ... }

    Description

    Flatten key-value pairs of maps within a map to a single map.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $map

    Map

    		Map— none
    $separator

    The separator between map keys.

    		String— none
    $_parent-key

    Used for recursion only.

    		String— none

    Returns

    Map —

    Returns flattened map.

    Example

    flatten(
  (
    "first": "content-first",
    "second": (
      "first": "content-second-first",
    )
  ),
  "--"
);
// ("first": "content-first", "second--first": "content-second-first")

    Throws

    • Duplicate key #{$key}.

    Requires

    • [variable] separator
    • [variable] separator
